HTML (Hypertext Markup Language) is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables. Your First 10 HTML Tags <.html> … <./html> — The root element. ... <.head> … <./head> — The document head. ... <.title> … <./title> — The page title. ... <.body> … <./body> — The page's content. ... <.h1> … <./h1> — A section heading. ... <.p> … <./p> — A paragraph. ... <.a> … <./a> — A link. ... <.img> — An image. The img element lets you insert images into your web pages.
