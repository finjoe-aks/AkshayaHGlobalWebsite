SQL


What is SQL ?

SQL is programming language used to control relational databases. SQL comprises both data definition and data manipulation languages. Using the data definition properties of SQL, one can design and modify database schema, whereas data manipulation properties allows SQL to store and retrieve data from database. SQL uses a set of commands to tell the DMBS what tasks and what type of tasks to do.


Types of Commands

There are two types of commands in SQL-Data Definition Language and Data Manipulation Language. Data Definition Language is used to Define the data or put the data in tables and databases. Data Manipulation Language is used to Manipulate or Modify and retrieve the data. For example selecting or deleting the data.
