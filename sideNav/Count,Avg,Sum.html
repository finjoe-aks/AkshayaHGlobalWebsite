Count(), Avg() and Sum() functions

The Count() function is used to find the number of rows in a table

The Avg() function is used to find the average of the specified column

The Sum() function is used to find the totalsum of the specified column

Syntax for Count() function

Select Count(columnname)
Fromtablename
Wherecondition ;

Null values will be ignored




Example for Count() function

The following statement will find the total number of footballers

Select Count(Ranking)
Fromfootballers ;

Null values will be ignored




Syntax for Avg() function

SelectAvg(columnname)
Fromtablename
Wherecondition ;

Null values will be ignored




Example for Avg() function

The following statement will find the average age of all footballers

SelectAvg(Age)
Fromfootballers ;

Null values will be ignored




Syntax for Sum() function

SelectSum(columnname)
Fromtablename
Wherecondition ;

Null values will be ignored




Example for Sum() function

The following statement will find the totalsum of the age of all footballers

SelectAvg(Age)
Fromfootballers ;

Null values will be ignored
