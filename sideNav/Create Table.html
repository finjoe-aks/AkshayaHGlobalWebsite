Create Table statement

The Create Table statement is used to create a database in SQL.

Syntax

Create Tabletablename (
column1 datatype,
column2 datatype,
column3 datatype,
...
);

datatype is the type of data going to be entered in the columns. For example the "Age" column's datatype will be "integer" or "varchar".



Example

The following statement will Create a Table called "footballers", with 5 columns called "Ranking","Name", "Age", "Club" and "Country" with datatypes "varchar", "varchar", "integer", "varchar" and "varchar". integer is written as "int".


Create Tablefootballers (
Ranking varchar(255)
Name varchar(255)
Age int(3)
Club varchar(255)
Country varchar(255)
);

You can give the maximum number of characters allowed in parenthesis after the datatype. This table is empty and has no data. To enter data you should use the "Insert Data" statement.
