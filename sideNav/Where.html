Where clause

The Where clause is used to filter the data in SQL by giving conditions. It is also used in statements such as Update and Delete to specify the data.

Syntax

Select column1, column2
From tablename
Where condition ;



Example for Numeric fields

The following statement will select all the footballers aged 30 from the table "footballers"

Select Age
From footballers
Where Age= 30 ;

Numeric fields should not be given in quotes. But all other fields should be given in quotes



Example for Text fields

The following statement will select all the footballers whose country is Argentina from the table "footballers"

Select Country
From footballers
Where Country= "Argentina" ;

symbols like
=, <, >, <=, >=, <>, Between, Like, In
can also be used with the "Where" clause to specify the data or add filter
