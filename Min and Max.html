Min() and Max() functions

The Min() function is used find the smallest value of the selected column

The Max() function is used find the largest value of the selected column

Syntax for Min() function

SelectMin(columnname)
Fromtablename
Wherecondition ;



Example for Min() function

The following statement will find the footballer with the smallest value in the "Age" column from the table "footballers"

SelectMin(Age)
Fromfootballers ;



Syntax for Max() function

SelectMax(columnname)
Fromtablename
Wherecondition ;



Example for Max() function

The following statement will find the footballer with the largest value in the "Age" column from the table "footballers"

SelectMax(Age)
Fromfootballers ;
