Delete statement

The Delete statement is used to delete data in a table

Syntax for deleting all the data from in a table

Delete Fromtablename ;



Example for deleting all the data in a table

The following statement will delete all the data in the table "footballers"

Delete Fromfootballers;



Syntax for deleting specified data in a table

The "Where" statement is used to specify what data to delete

Delete Fromtablename
Where condition;



Example for deleting specified data in a table

The following statement will delete all the footballers's data in the table "footballers" whose age is 33

The "Where" statement is used to specify what data to delete

Delete Fromfootballers
Where Age= "33" ;
