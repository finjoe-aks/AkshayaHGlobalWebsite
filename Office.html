Spreadsheets, Mailmerging and Word


What is a Spreadsheet?

A spreadsheet is a file that exists of cells in rows and columns and can help arrange, calculate and sort data. Data in a spreadsheet can be numeric values, as well as text, formulas, references and functions. Excel is one of the most widely used spreadsheet applications. It is a part of Microsoft Office suite.

Some of it's uses are:

  • Business Data Storage.

  • Accounting and Calculation Uses.

  • Budgeting and Spending Help.

  • Assisting with Data Exports.

  • Data Sifting and Cleanup.

  • Generating Reports and Charts.

  • Business Administrative Tasks.

What is Mail Merging?

Mail merge is a feature within most data processing applications that enables users to send a similar letter or document to multiple recipients. It enables connecting a single form template with a data source that contains information about the recipient's name, address and other predefined and supported data.

Components of Mail Merging:

There are three documents that are involved in the mail merge process: the main document, the data source, and the merged document.

Word

Microsoft Word or MS Word is a graphical word processing program that users can type with. It is made by the computer company Microsoft. Its purpose is to allow users to type and save documents. Similar to other word processors, it has helpful tools to make documents.

What is in MS Word

Microsoft Word allows you to create professional-quality documents, reports, letters, and résumés. Unlike a plain text editor, Microsoft Word has features including spell check, grammar check, text and font formatting, HTML support, image support, advanced page layout, and more.
