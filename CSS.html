CSS(Cascading Style Sheets) is used for designing webpages. It describes the way HTML elements should be displayed on the webpage.

There are 3 ways to write CSS

  • Inline CSS
  • External CSS
  • Embedded CSS

    • In inline CSS the code is written inside the tag itself by using to style tag. Inline css are mostly used by beginners because it's easy for them, but it gets really messy when a there's a lot of code present. That's when we use external CSS. External CSS is written is a seperate file or folder made for CSS alone. A number of other HTML files can be attatched to the .css file by using the tag: link rel="stylesheet" href="name of stylesheet". Embedded CSS is where we write the stylings inside the style tag in the head tag. This type of styling is more organized than inline CSS but still the best method is to use external CSS.



    Now, let's quickly look at the parts of a CSS style.

    The selector is the area or place where the CSS should be applied. The property defines what type of stylings are to be displayed. And lastly the value is the amount of the property to be displayed. The colors such as green, blue, etc are all values of the property color.
