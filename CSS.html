What is CSS ?

CSS(Cascading Style Sheets) is used for designing webpages. It describes the way HTML elements should be displayed on the webpage.

There are 3 ways to write css-Inline, External and Embedded CSS.

  • In inline CSS, the code is written inside the tag itself by using to style tag. Inline css are mostly used by beginners because it's easy for them.

  • It gets really messy when a there's a lot of code present. That's when we use external CSS. External CSS is written is a seperate file or folder made for CSS alone. A number of other HTML files can be attatched to the .css file by using the tag: link rel="stylesheet" href="name of stylesheet".

  • Embedded CSS is where we write the stylings inside the style tag in the head tag. This type of styling is more organized than inline CSS but still the best method is to use external CSS.



Now, let's quickly look at the parts of a CSS style.

The selector is the area or place where the CSS should be applied. The property defines what type of stylings are to be displayed. And lastly the value is the amount of the property to be displayed. The colors such as green, blue, etc are all values of the property color.

CSS colors:

Using colors properly is a pretty hard job even for a professional web designer. Colors are fairly simple to use but they play a main role in displaying the website. A single color used properly will definitely look better than a multiple colors used improperly. So it's important that we learn how we use colors and the different ways to write them.

Names:
Typing down the actual names of the color like "Orange", "Red","SeaGreen", etc. Color details like shades, saturation, lightness may not be availabe while using this method.
HEX values:
Hexadecimal color codes are written with a # in front of them to make them work. Examples include #ff0000(red), #40ff00(green), etc.
RGB values:
RGB values are written in terms of the amounts of red, green, and blue in the color. For example, rgb(255, 0, 144) will give a shade of pink.
HSL values:
HSL stands for Hue, Saturation, Lightness and by increasing or decreasing the following values we get dfferent colors. Example: hsl(30Â°, 100%, 40%) will give a dull orange color.

There are thousands of values for each values given above. And it's not necessary to memorize each one of them. However you have to know atleast a few easy color codes to keep up with today's world. For viewing more color codes visit Google's Color picker and you can adjust the settings a bit to achieve the color you want. Text editors like VS code have built in color pickers so you don't have to type in a query on google everytime.

CSS fonts:

Fonts have a relatively large role in styling today's text. Fonts are present everywhere from word documents to coded websites. Old fonts like serif are not very attractive in modern websites. Fonts that are plain and simple like sans-serif are look more pleasing and attractive. Fonts are really easy to use and they are not complex like colors. Let's quickly look at how we use them.

Font-family:
Font-families are the different types of fonts we regularly see in word documents. These fonts include serif, sans-serif, roboto, montserrat, Arial and hundreds of more fonts.
Font-size:
The font-size property specifies the size of the text displayed on the screen. The different values used are pixels, percentages, etc. Values like large, xx-small, small, etc. can also be used to specify font-sizes.
Font-weight:
Font-weight indicates the boldness of the text. There are values such as bold, lighter, bolder, etc. and numerical values from 100 to 900.
font-style:
The font-style property allows you make the text appear italicized. There are only three values for this property: normal, italic and oblique.
font-variant:
The font-variant property allows you to make the text lowercase. Simply use font-variant:small-caps to transform the selected letters to lowercase. Unfortunately there's no all-caps option in this property. For all caps type in text-transform:uppercase to make the text appear in all-caps.

Unlike CSS colors, there are only a few properties and values for fonts. As there are not much of numerical or decimal values like in colors, it makes it even more easier to remember and use the values. Although, the difficult part is that fonts do not change the document as colors do and so focusing on fonts more than colors is not worth it.

Widths and Heights:

The width and height attributes are two of the most important properties of a page. Even the height of an element can be ignored sometimes, but the width is very important. We've all seen and used the scrollbar atleast once in our life. The scroll bar is used when the height of the content exceeds the screen height itself; that's normal to all of us. But scrolling sideways is not very normal. That's why the width is important. If the width is not specified, then a sidebar will be auomatically added to display the content, which of course irritates people.

Width:
The width of an element such as an image, a div, a block, etc. This property can be specified using values like pixels and percentages.
Height:
The height attribute specifies the height of a html element. Height has values similar to the width property but this property is mostly ignored. If the height of an element is not specified, then the page will become lengthier as you keep adding content.
Min- and max- values:
Min and max properties are used to write a very specific size to an element. For example, "min-height:100px;" - The minimum height given is 100px. Which means the element can only grow taller but no shorter than 100 pixels. The same process works for max-height too but in the opposite way.

By now we should be familiar with width and height. The min- and max- values are similar to the normal attributes but very specific. These minimum and maximum values are mostly used to write responsive CSS. The same CSS for a desktop can't be used for a mobile phone. The usual laptop screen is about 1300 pixels in length. The CSS used for a normal laptop will of course look normal, but on a giant screen (like 2500px in length) the sizes and spaces will look very different. That's why we specify the width using min and max-width.
