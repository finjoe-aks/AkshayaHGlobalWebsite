And, Or and Not operators

The And operator displays a record if all the conditions seperated by And are true

The Or operator displays a record if either of the conditions seperated by Or is true

The Not operator displays a record if all the conditions are not true

All these operators are used combined with the Where clause

Syntax for And

Select column1, column2, ...
From tablename
Where condition1 And condition2 And condition3 ... ;



Example for And

The following statement will select all footballers from the table "footballers" whose country is Argentina and club is Barcelona

Select * From footballers
Where Country= "Argentina" And Club= "Barcelona" ;



Syntax for Or

Select column1, column2, ...
From tablename
Where condition1 And condition2 Or condition3 ... ;



Example for Or

The following statement will select all footballers from the table "footballers" whose country is either Argentina or Spain

Select * From footballers
Where Country= "Argentina" Or Country= "Spain" ;



Syntax for Not

Select column1, column2, ...
From tablename
Where Not condition ;



Example for Not

The following statement will select all footballers from the table "footballers" whose country is not Argentina

Select * From footballers
Where Not Country= "Argentina" ;



You can also combine the And, Or and Not operators
